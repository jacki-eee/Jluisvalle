The College experience Game

Software Engineering Group Project

  • Project Overview:

    This project aims to develop a game to entertain students, offering a fun solution to boredom. Designed specifically for students, it provides quick, engaging distractions that can serve as a fidget game during class.
    While alternatives like the Snake game, Concord, and platforms such as Coolmath and Google exist, our game stands out by incorporating humor and relatable commentary on student life.
    This game is compelling because it's not just another game, but one created with students in mind, offering gameplay tailored to their needs. The top-level objective is to create a simple, accessible game with a small scope. Its key differentiator is its focus on the student experience and commentary.


    Scope of Work:

    This project involves the design and development of The College Experience Game.
    Key tasks include:

    • Game Development: Implement movement, weapons, power-ups, and inventory management.
    • User Experience: Design an intuitive interface.
    • Backend & Data Management: Enable user authentication and progress tracking.
    • Testing & Optimization: Test gameplay, and fix bugs.
    • Engineering Process: Use Scrum methodology for development
